라이브러리사용하기
라이브러리만들기

c에서의 라이브러리 함수로 작성되었다.
int add( int x, int y) {}
int sqrt( int x){}
int abs( int x) {}



자바에서 라이브러리는 클래스 단위로 작성되었다.
하나의 함수를 작성하고 싶어도  class가 필요하다
class Math{
      public static  int add( int x, int y){}
      public static  int sqrt ( int x){}
      public static  int abs ( int x) {}
}


Math클래스를 사용하는 방법의 의미로 매서드라는 표현을 한다.
(기존의 함수와 작성방법은 동일하다)

사용법은 
int   result  =Math.abs(-7);
double result2  =  Math.sqrt(4);
