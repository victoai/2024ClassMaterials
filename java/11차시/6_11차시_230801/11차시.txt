-- static 정리

   static은 매서드와  static변수는 다른 의미를 갖는다. !

   static 매서드  (  일반적인 함수 ,  어떤 클래스에 소속된 기능, 함수 의미) 사용법 : 클래스명.매서드명()
   static이 아닌 매서드 ( 클래스형으로 만들어진 객체를 다루는 기능 )  객체생성, 즉 new 이후부터  객체명.매서드()


   객체변수 (인스턴스변수)   :  new할 때마다 변수가 만들어짐 (객체가 만들어짐)
    static변수  : 해당 클래스에서 같이 사용하는 전역변수이다
