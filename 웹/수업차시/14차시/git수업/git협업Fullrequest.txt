upstream      :   원본 저장소(협업 할 github 저장소 ) : 조장님 리포지토리
origin           :  나의 github 저장소
<단축이름>  :임의로 작성해도 된다.


원격저장소 추가하기  (원격저장소  더하기 )
git remote add  origin              <url>  (내원격저장소 주소)
git remote add  upstream         <url>  (원본원격저장소 주소) :조장님꺼
                      (단축이름)       (원격저장소위치)


#원격저장소 삭제하기
git remote rm origin


#원격저장소 조회하기
git  remote -v

#원격저장소에 밀어넣기(업로드)
git push -u origin  main
git push   ( 이것만 해도 됨)

#원격저장소와 동기화하기 
git pull  upstream

#내원격저장소에 push하기
git push origin main
