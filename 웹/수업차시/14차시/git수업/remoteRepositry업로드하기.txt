내컴퓨터 작업디렉토리에 있는 작업들을 
원격의 컴퓨터에 밀어넣기 (업로드하기) 


local repository 원격저장소에 밀어 넣기 
remote  repository


github 원격저장소에 밀어넣기 ( 원격저장소로 사용하기)

--원격저장소 목록보기
git remote 

--원격저장소 추가하기
git remote add origin  https://github.com/victoai/first.git
                     
         -원격저장소별칭( origin) , 다른이름을 사용해도 된다.
         -원격저장소 주소

--원격저장소 밀어넣기, 업로드 
  git push -u origin main
  
  -u의 의미 뒤의 origin과 main branch를 기본으로 하겠다
  다음부터는 뒤의 내용을 생략할 수 있다.


주의!!
원격이 바뀐경우
full  & push 한다
-먼저 full하고 push한다.
